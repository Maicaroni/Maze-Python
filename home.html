How to Play

Help the bunny find a way out of the maze using programming commands in the console! But be careful, the bunny only moves when you write correct code using a for loop.

Your objective is to guide the bunny to the maze's exit without losing all hearts.

  • Difficulty Levels: Choose a difficulty (Easy, Medium, Hard). Once selected, click Start to generate a maze matching your chosen difficulty.
  • Hearts: You start with 3 hearts. Every time the bunny hits a wall, you lose a heart. If all hearts are lost, the bunny is sent back to the starting point.
  • Info: Click the info icon to see a list of commands if you need help.
